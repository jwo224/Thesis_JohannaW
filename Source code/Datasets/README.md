# Datasets

## `YOLO_Trolley_Datasets/`

Contains the YOLO trolley training material:

- training notebooks
- lighting robustness report/notebook
- trained model runs and weights
- `YoloLabel_v1.2.1`
- base YOLO weights

These files are separated from the ROS2 workspaces so the workspaces stay clean.
ROS packages still keep the model weights they load directly at runtime.
