# Mission Baseline Results

This folder contains an older copy of `baseline_mission_results.csv`.

The active default output folder for new baseline runs is:

```text
~/Nursing-Home-Robot/mission_baseline_results
```

See:

```text
~/Nursing-Home-Robot/mission_baseline_results/README.md
```

New baseline runs include stage-prefixed ArUco/Gazebo columns there:

- `side_align_*` for the outside/pre-entry ArUco alignment.
- `fine_align_*` for the under-trolley ArUco alignment before attach.
