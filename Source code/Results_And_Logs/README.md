# Results And Logs

This folder contains experiment outputs moved out of the ROS2 workspaces:

- ArUco reference logs
- trolley reference logs
- YOLO detection logs
- mission run logs
- BLE/Wi-Fi distance checks
- FSR/elevator CSV logs
- simulation mission result folders
- the `logsyolo` result bundle

These files are useful for thesis traceability, but they are not required to
build the ROS2 workspaces.
